### Peer.js Server

#### Connecting to the server from client PeerJS:

```
const myPeer = new Peer(undefined, {
  host: '/',
  port: '3001'
})

```

### Run 

1- Open Project

2- Run `peerjs --port 3001` on one terminal to run peer js server

3- Run `npm start` to run node server.js
